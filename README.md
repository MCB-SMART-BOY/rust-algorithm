## 🚀 快速开始

### 环境要求

- Rust 1.70+ (按照下面做就好)
- Cargo (Rust 包管理器，随 rustup 安装)

### 安装 Rust

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh // linux 发行版可以直接包管理器安装的（应该没有发行版官方仓库中没这个吧）
rustup default stable // 这里开始自动下载 rust 相关的工具链（真tm方便）
```

### 克隆项目

```bash
git clone https://github.com/MCB-SMART-BOY/rust-algorithm.git
cd rust-algorithm
```

## 🧠 刷题思路

### 如何使用本项目

1. **选择题目**: 从题目列表中选择一道分类下的题目
2. **独立思考**: 先尝试自己分析题意，设计算法思路
3. **编写代码**: 使用 Rust 实现算法
4. **运行验证**: 运行代码验证思路正确性
5. **复盘优化**: 分析代码的时间和空间复杂度，思考优化方案

### 刷题建议

- **按分类刷**: 同一类型的题目连续刷，更好地掌握解题模式
- **先易后难**: 从简单题开始，逐步过渡到困难题
- **定期回顾**: 定期复习已完成的题目，加深理解
- **思路复盘**: 重点关注思路错误的地方，分析原因并纠正
